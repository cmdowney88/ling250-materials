# Data Files

This folder contains data files used in course demos and exercises.

## Files to Download

Some files cannot be included in this repository due to copyright restrictions. Please download them from Blackboard and place them in this folder.

### Night_Vale.txt

A transcript of the Welcome to Night Vale podcast, used for text processing examples throughout the course.

**To obtain:**
1. Go to Blackboard → LING 250 → Course Materials
2. Download `Night_Vale.txt`
3. Place it in this folder (`data/Night_Vale.txt`)

**Verify it's in the right place:**
```bash
ls data/Night_Vale.txt
```

If you see the file listed (not "No such file or directory"), you're all set.
