# LING 250/450: Data Science for Linguistics

Course materials for LING 250/450 at the University of Rochester (Spring 2026).

## Setup

1. **Clone this repository:**
   ```bash
   git clone https://github.com/cmdowney88/ling250-materials.git
   cd ling250-materials
   ```

2. **Download additional data files:**
   Some data files used in this course cannot be distributed in this repository due to copyright. See `data/README.md` for instructions on obtaining these files.

3. **Install required packages:**
   Make sure you have Anaconda installed (see the setup guide on the course website), then:
   ```bash
   conda activate base  # or your preferred environment
   pip install nltk
   ```
   Additional packages will be introduced as needed throughout the semester.
